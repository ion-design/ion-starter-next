# Brand Guidelines

This skill provides company-specific brand identity context.

## How to use

1. Check to see if a /brand page exists in the app that goes over the company's brand guidelines (colors, typography, design language, component patterns, voice, strategy).
2. Otherwise, read `company-docs/brand-guidelines.md` for the company's full brand guidelines (colors, typography, design language, component patterns, voice, strategy).
3. Apply those guidelines to all design decisions, image generation prompts, and visual artifacts.
4. If neither `company-docs/brand-guidelines.md` nor /brand exist, note that no brand guidelines are available and proceed with best judgment using clean, professional defaults.
